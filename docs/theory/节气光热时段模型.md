# 节气光热时段模型

## 1. 总公式

```text
人格类型 = 节气令长期认知行为偏好 × 日内光热行动入口 × 节律交互
```

本体系的当前目标，是逐步摆脱传统元素系统，形成以现代自然节律为核心的独立表达。

当前主语是：

```text
二十四节气令
×
十二个日内光热时段
```

类型命名直接写成：

```text
节气 · 日内时段
```

例如：

```text
夏至·日出
秋分·日落前
冬至·深夜
立春·破晓前
```

## 2. 年度端：节气令

节气令描述一个人长期如何理解世界、整理经验、处理阶段变化与选择行动方向。

月份、季度、季节仍可作为粗粒度背景，但当前体系以节气令为精密年度锚点。

| 层级 | 用途 | 精度 |
|---|---|---|
| 季节 | 描述年度大趋势 | 粗 |
| 季度 | 描述较长周期倾向 | 粗 |
| 月份 | 描述社会日历段 | 中 |
| 节气令 | 描述太阳黄经 15° 的年度相位 | 细 |

## 3. 日内端：光热时段

日内光热时段描述出生后第一段世界经验如何影响行动入口。

它优先使用地方太阳时，并关注：

- 日出；
- 早晨活动启动；
- 正午前光照显明；
- 午后高热；
- 下午道路与信息通畅；
- 日落前边界增强；
- 日落后余热未散；
- 夜初躁动；
- 入夜沉降；
- 深夜潜感；
- 黎明前最低温；
- 破晓前预感。

| 编码 | 时段 | 动作名 | 核心定义 |
|---|---|---|---|
| D01 | 日出 | 光明展开 | 生命相信世界继续打开 |
| D02 | 早晨启动 | 活动升温 | 社会活动、劳动、学习开始启动 |
| D03 | 正午前 | 清明显现 | 光照清明，自我表达增强 |
| D04 | 午后高热 | 热量冲顶 | 温度继续累积，行动阈值降低 |
| D05 | 下午通路 | 路径打开 | 信息、道路、交换变顺 |
| D06 | 日落前 | 边界裁断 | 影长光斜，结算与判断增强 |
| D07 | 日落 | 残照落幕 | 光被切断，余热与终局感并存 |
| D08 | 夜初 | 余热躁动 | 外界关门，身体热度仍在撞门 |
| D09 | 入夜 | 意识沉降 | 活动止息，经验沉入记忆 |
| D10 | 深夜 | 潜感活跃 | 梦性、暗线、隐秘感增强 |
| D11 | 黎明前 | 最低温封存 | 冷静、防御、低表达、高耐受 |

## 5. 节律交互

新版关系不再以传统元素关系为主，而用节律动作描述两端如何相遇。

| 关系 | 含义 |
|---|---|
| 同向放大 | 年度端与日内端处在同一节律方向 |
| 顺势接力 | 年度端自然推动日内端，形成接力 |
| 逆向回流 | 日内端把年度端拉回前一节律 |
| 热度成形 | 高热、显化、表达压力被压成作品、标准、结构 |
| 标准回压 | 边界、标准、结果意识反向审查热度 |
| 显化回藏冲突 | 外放、加速、显现与回收、延迟、潜入相冲 |
| 裁剪生长 | 边界意识对生长过程形成修枝 |
| 生长遇界 | 生长冲动遭遇边界与结算 |
| 过渡转译 | 两端不同，但一端处在转换带 |
| 并行调和 | 两端弱冲突、弱推进，形成并行模块 |

## 6. 示例：夏至·日出

```text
夏至·日出
```

夏至描述年度端的“极光显明”：倾向把事物推到可见中心，追求清晰显现、公开表达、中心位置与高光照判断。

日出描述日内端的“光明展开”：行动入口偏恢复、展开、再生与重新开始。

合起来，这一类型的核心是：

```text
长期倾向把事物推向可见中心，并以日出式的恢复、启动、展开方式进入行动。
```

## 7. 判读边界

本模型只描述：

- 长期认知行为偏好；
- 行动入口；
- 感知入口；
- 判断方式；
- 精神动力结构；
- 节律错位或同步。

它不直接判定现实成败或固定命运。
