# 节气光热时段模型

## 1. 总公式

```text
人格底色 = 节气令长期认知行为偏好 × 日内光热人格底色 × 节律交互
```

本体系的当前目标，是逐步摆脱传统元素系统，形成以现代自然节律为核心的独立表达。

这里需要明确一条边界：

```text
日内光热时段不是行动入口，而是人格底色。
```

它描述的是出生时一日光暗、温度、热量滞后与社会活动节律在心理底层留下的体感结构。

当前主语是：

```text
二十四节气令
×
十二个日内光热时段
```

类型命名直接写成：

```text
节气 · 日内时段
```

例如：

```text
夏至·日出
秋分·日落前
冬至·深夜
立春·破晓前
```

## 2. 年度端：节气令

节气令描述一个人长期如何理解世界、整理经验、处理阶段变化与选择方向。

月份、季度、季节仍可作为粗粒度背景，但当前体系以节气令为精密年度锚点。

| 层级 | 用途 | 精度 |
|---|---|---|
| 季节 | 描述年度大趋势 | 粗 |
| 季度 | 描述较长周期倾向 | 粗 |
| 月份 | 描述社会日历段 | 中 |
| 节气令 | 描述太阳黄经 15° 的年度相位 | 细 |

## 3. 日内端：光热人格底色

日内光热时段描述出生时那一段昼夜环境如何形成底层世界印象。

它优先使用地方太阳时，并关注：

- 日出后的光明展开；
- 早晨的活动升温；
- 正午前的光照清明；
- 午后高热的热量压强；
- 下午道路与信息通畅；
- 日落前边界增强；
- 日落后余热未散；
- 夜初躁动；
- 入夜沉降；
- 深夜潜感；
- 黎明前最低温；
- 破晓前预感。

这些时段不直接说明一个人怎样行动。它们描述一个人最早感受到的世界温度、光暗边界、身体节律和心理底色。

## 5. 节律交互

新版关系不再以传统元素关系为主，而用节律动作描述年度端与日内人格底色如何叠合。

| 关系 | 含义 |
|---|---|
| 同向放大 | 年度端与日内端处在同一节律方向 |
| 顺势接力 | 年度端自然推动日内端，形成接力 |
| 逆向回流 | 日内端把年度端拉回前一节律 |
| 热度成形 | 高热、显化、表达压力被压成作品、标准、结构 |
| 标准回压 | 边界、标准、结果意识反向审查热度 |
| 显化回藏冲突 | 外放、加速、显现与回收、延迟、潜入相冲 |
| 裁剪生长 | 边界意识对生长过程形成修枝 |
| 生长遇界 | 生长冲动遭遇边界与结算 |
| 过渡转译 | 两端不同，但一端处在转换带 |
| 并行调和 | 两端弱冲突、弱推进，形成并行模块 |

## 6. 示例：夏至·日出

```text
夏至·日出
```

夏至描述年度端的“极光显明”：倾向把事物推到可见中心，追求清晰显现、公开表达、中心位置与高光照判断。

日出描述日内端的“光明展开”：世界正在打开，底层体感偏恢复、展开、再生与重新开始。

合起来，这一类型的核心是：

```text
长期倾向把事物推向可见中心，同时拥有日出式的恢复、展开、再生人格底色。
```

## 7. 判读边界

本模型只描述：

- 人格底色；
- 长期认知行为偏好；
- 昼夜体感结构；
- 感知方式；
- 判断方式；
- 精神动力结构；
- 节律错位或同步。

它不直接判定现实成败或固定命运。
