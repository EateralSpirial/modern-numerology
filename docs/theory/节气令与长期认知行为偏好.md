# 节气令与长期认知行为偏好

## 1. 定义

节气令，是指出生时太阳所处的二十四节气段。

在现代命理学中，节气令不应被理解为欲望模式，也不应被粗略等同于月份或季度。它更适合描述一个人长期重复使用的认知行为偏好。

```text
节气令 = 太阳年相位在人格中的长期认知行为刻印
```

## 2. 为什么不用“欲望模式”作为主语

“欲望模式”容易把年度端解释得过于心理化，也容易把季节相位误读成简单的追求对象。

年度端真正稳定的东西，更像一种长期认知动作：

- 初春到春末：偏向开局、修复、唤醒、连接、伸展；
- 初夏到盛夏：偏向启动、增量、显明、表达、任务密度；
- 初秋到深秋：偏向总结、分割、衡量、归类、裁断；
- 初冬到深冬：偏向回藏、记忆、保存、潜伏、等待反转。

因此，新版表述统一为：

```text
长期认知行为偏好
```

## 3. 月份、季度、季节的定位

月份、季度和季节仍然可用，但它们只能作为粗粒度层级。

| 层级 | 作用 | 精度 |
|---|---|---|
| 季节 | 描述年度大趋势 | 粗 |
| 季度 | 描述较长周期的认知倾向 | 粗 |
| 月份 | 描述约 30 天的社会日历段 | 中 |
| 节气令 | 描述太阳黄经 15° 的年度相位 | 细 |

所以新版体系应以节气令作为年度主轴。

## 4. 秋分示例

秋分位于太阳黄经 180°，昼夜再次平分，年度节律进入平衡、分割、度量与裁判位。

它的长期认知行为偏好不是“想要什么”，而是倾向于怎样处理经验：

- 总结经验；
- 分割对象；
- 衡量轻重；
- 归类材料；
- 建立对称关系；
- 设定边界；
- 判断比例；
- 完成阶段性结算。

所以秋分的核心句可以写成：

```text
秋分倾向于把混杂经验切分、衡量、归类，并转化为可判断的结构。
```

## 5. 节气令与日内光热段的组合

年度端给出长期认知行为偏好，日内端给出人格底色中的昼夜体感层。

```text
节气令：长期如何理解、分类、推进或保存经验
日内段：出生时一日光暗、温度、热量滞后与社会活动节律如何形成底层世界印象
```

例如：

```text
秋分·日落前
```

可以读为：

> 长期认知上倾向总结、分割、衡量、归类；日落前的人格底色也倾向边界、判断、切割、结算。两端形成强烈的结构判断、阶段清算与标准化倾向。

## 6. 判读边界

节气令只描述长期认知行为偏好。它不能单独决定现实成败或固定命运。

完整判断仍需结合：

- 出生地气候与纬度；
- 地方太阳时；
- 月相；
- 孕期季节路径；
- 家庭与时代现实变量；
- 个体后天选择。
