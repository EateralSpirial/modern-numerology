# 五行性格相位模型

## 1. 总公式

```text
人格类型 = 节气令长期认知行为偏好 × 日内光热行动入口 × 五行相位交互
```

本体系已经完成一次主语重构：年度端不再以十二月支、十二个月度相位或农历月份作为主要解释层，而改用二十四节气令。

当前理论主语是：

```text
一年中的二十四节气令
×
一天中的光热相位
```

前者描述长期认知行为偏好，后者描述行动入口、感知入口和判断入口。

## 2. 为什么用节气令替代月份

月份、季度和季节仍然有意义，但它们粒度较粗。它们更适合描述大范围长期倾向，例如春季偏生发、夏季偏显化、秋季偏收束、冬季偏回藏。

若要建立更精密的人格模型，年度端应该使用二十四节气令：

```text
一个节气 = 太阳黄经推进 15° 后形成的年度相位段
```

因此，节气令比“月份”更接近自然理学锚点。它描述的重点也需要调整为：

```text
旧表述：欲望模式底色
新表述：长期认知行为偏好
```

例如，秋分令的核心不是某种欲望，而是一种长期认知动作：

- 总结；
- 分割；
- 衡量；
- 归类；
- 建立对称关系；
- 判断边界与比例。

## 3. 节气令系统：土作为相位缓冲

节气令系统中，土不再被视为与木、火、金、水并列的独立主季节。土主要承担：

- 前后两个季节的连接；
- 向下一个相位转化；
- 孕育、含藏、缓冲；
- 将旧相位中的资源转译为新相位的燃料。

所以年度端的主运动应写成：

| 年度主运动 | 节气范围 | 长期认知行为倾向 |
|---|---|---|
| 木 | 立春—谷雨 | 寻找开局、伸展路径、连接资源、使事物生长 |
| 火 | 立夏—大暑 | 显化、推进、表达、组织任务、提高现实热度 |
| 金 | 立秋—霜降 | 复盘、分割、判断、归类、定型、清算 |
| 水 | 立冬—大寒 | 回藏、记忆、内省、保存、潜伏、等待反转 |

土是所有节气交界处的相位缓冲、承载和转化介质。

## 4. 日内系统：使用新的光热曲线

日内系统不能继续照搬十二地支。一天的真实结构是：

- 正午太阳最高；
- 午后温度最高；
- 黎明前温度最低；
- 日落后光已断，但热仍留；
- 刚入夜并不立刻入水，而有余热、躁动和执着；
- 日出前有“光未至而阳先动”的先知性。

因此日内十二相位应为：

| 序号 | 日内相位 | 属性 | 核心定义 |
|---:|---|---|---|
| 1 | 强劲木 | 木 | 日出后迎来完整白昼，生命相信展开 |
| 2 | 朝气火 | 火 | 世界启动，行动与社会性全面升温 |
| 3 | 热情火 | 火 | 光照清明，自我表达与热情增强 |
| 4 | 爆火 | 火 | 温度冲顶，冲动、爆发、强烈表达 |
| 5 | 疏通金 | 金 | 热退而路通，信息、技术、交换变灵活 |
| 6 | 裁断金 | 金 | 日斜将落，边界、判断、结算增强 |
| 7 | 残照金 | 金兼余火 | 日落天暗，终局感、余热与执念并存 |
| 8 | 躁动金 | 燥金 | 夜初仍热，残火撞击边界 |
| 9 | 沉降水 | 水 | 活动止息，意识、情绪、记忆下沉 |
| 10 | 感性水 | 水 | 深夜潜流，梦性、情绪、隐秘性增强 |
| 11 | 冰冷水 | 水 | 黎明前最低温，冷静、封存、防御 |
| 12 | 先知木 | 冷木 | 光未至而阳先动，预感、等待、先发意识 |

## 5. 五行交互机制

新版关系不再使用传统六合、六冲、三合等地支关系作为主逻辑，而使用五行相位关系：

| 关系 | 含义 |
|---|---|
| 同气放大 | 节气令与日内段主气一致，长期认知偏好与行动入口同频 |
| 顺势推进 | 年度端自然推动日内端，形成较顺的推进 |
| 逆势回流 | 日内端把年度端拉回前一相位，形成复盘、补根或回撤 |
| 炼化成器 | 火对金形成助成，把热度压成结构、标准、作品 |
| 结构回火 | 金反向要求火服从标准，形成高审查和高定型 |
| 水火时间冲突 | 显化时间与回流时间相冲，温差最大 |
| 裁断生长 / 生长遇裁 | 金木之间的切割、生长、修枝关系 |
| 相位转译 | 交界气负责吸收、延迟、孕育、转化异质相位 |
| 并行调和 | 二者并行，弱冲突、弱生发，适应性较强 |

## 6. 判读边界

本模型优先描述：

- 长期认知行为偏好；
- 行动入口；
- 感知入口；
- 判断方式；
- 内耗机制；
- 相位错位或同步。

它不直接判定现实财富、职业等级、婚恋结果、疾病、寿夭或固定命运。
