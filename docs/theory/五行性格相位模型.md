# 五行性格相位模型

## 1. 总公式

```text
人格类型 = 月度季节五行底色 × 日内光热五行底色 × 五行相位交互
```

本体系已经不再把人格模型挂靠到“月支 × 时支”的旧框架上。传统地支只保留为历史索引或转换表，当前理论主语是：

```text
一年中的季节相位
×
一天中的光热相位
```

前者描述欲望底色，后者描述行动入口、感知入口和判断入口。

## 2. 月度系统：弱化土

月度系统中，土不再被视为与木、火、金、水并列的主季节。土主要承担：

- 前后两个季节的连接；
- 向下一个相位转化；
- 孕育、含藏、缓冲；
- 将旧相位中的资源转译为新相位的燃料。

因此，旧系统中的四个“土月”应改写为：

| 月度相位 | 旧索引 | 新理解 |
|---|---|---|
| 孕木寒水 | 冬末土 | 水向木转化，寒水中孕木 |
| 蓄火木 | 春末土 | 木向火转化，木气蓄满将燃 |
| 炼金火 | 夏末土 | 火向金转化，火炼金、热成器 |
| 藏水金 | 秋末土 | 金向水转化，裁断后闭藏入水 |

所以月度系统里的土是**相位转换介质**，不是独立主属性。

## 3. 日内系统：使用新的光热曲线

日内系统不能继续照搬十二地支。一天的真实结构是：

- 正午太阳最高；
- 午后温度最高；
- 黎明前温度最低；
- 日落后光已断，但热仍留；
- 刚入夜并不立刻入水，而有余热、躁动和执着；
- 日出前有“光未至而阳先动”的先知性。

因此日内十二相位应为：

| 序号 | 日内相位 | 属性 | 核心定义 |
|---:|---|---|---|
| 1 | 强劲木 | 木 | 日出后迎来完整白昼，生命相信展开 |
| 2 | 朝气火 | 火 | 世界启动，行动与社会性全面升温 |
| 3 | 热情火 | 火 | 光照清明，自我表达与热情增强 |
| 4 | 爆火 | 火 | 温度冲顶，冲动、爆发、强烈欲望 |
| 5 | 疏通金 | 金 | 热退而路通，信息、技术、交换变灵活 |
| 6 | 裁断金 | 金 | 日斜将落，边界、判断、结算增强 |
| 7 | 残照土金 | 土金 | 日落天暗，终局感、执着与守持并存 |
| 8 | 躁动土金 | 土金 | 夜初仍热，欲望残火撞击边界 |
| 9 | 沉降水 | 水 | 活动止息，意识、情绪、记忆下沉 |
| 10 | 感性水 | 水 | 深夜潜流，梦性、情绪、隐秘性增强 |
| 11 | 冰冷水 | 水 | 黎明前最低温，冷静、封存、防御 |
| 12 | 先知土木 | 土木 | 光未至而阳先动，含藏、预感、等待 |

这里的土具有性格意义：它不是季节主轴，而是**在黑夜、余热、等待、含藏中的执着与承载**。

## 4. 五行交互机制

新版关系不再使用传统六合、六冲、三合等地支关系作为主逻辑，而使用五行相位关系：

| 关系 | 含义 |
|---|---|
| 同气放大 | 月度端与日内端主气一致，欲望与行动入口同频 |
| 顺势推进 | 月度端自然推动日内端，形成较顺的推进 |
| 逆势回流 | 日内端把月度欲望拉回前一相位，形成复盘、补根或回撤 |
| 炼化成器 | 火对金形成助成，把热度压成结构、标准、作品 |
| 结构回火 | 金反向要求火服从标准，形成高审查和高定型 |
| 水火时间冲突 | 显化时间与回流时间相冲，温差最大 |
| 裁断生长 / 生长遇裁 | 金木之间的切割、生长、修枝关系 |
| 土性承载 / 土性转译 | 土负责吸收、延迟、孕育、转化异质相位 |
| 并行调和 | 二者并行，弱冲突、弱生发，适应性较强 |

## 5. 判读边界

本模型优先描述：

- 人格底色；
- 欲望生成方式；
- 行动入口；
- 感知入口；
- 判断方式；
- 内耗机制；
- 相位错位或同步。

它不直接判定现实财富、职业等级、婚恋结果、疾病、寿夭或固定命运。
