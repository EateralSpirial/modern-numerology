# 二十四节气命理学：节气令体系

## 1. 基本定义

二十四节气命理学，是以太阳在黄道上的位置为年度气机锚点的现代命理描述系统。

```text
每一个节气 = 太阳黄经每推进 15° 后形成的年度气机段
```

在本体系中，传统“月令”改写为：

```text
节气令 = 出生时太阳所处的节气段
```

节气令描述的重点，是出生时所处太阳年相位带来的长期认知行为偏好。

## 2. 五行原则

二十四节气不再机械套入传统十二月支。

| 季节段 | 主五行 | 核心过程 | 长期认知行为偏好 |
|---|---|---|---|
| 立春—谷雨 | 木 | 生发、破土、伸展 | 寻找开局、连接资源、使事物生长 |
| 立夏—大暑 | 火 | 显化、升温、扩张 | 推进、表达、组织任务、提高现实热度 |
| 立秋—霜降 | 金 | 收束、分割、判断 | 总结、分割、衡量、归类、清算 |
| 立冬—大寒 | 水 | 回藏、沉降、封存 | 记忆、内省、保存、潜伏、等待反转 |

土不单独作为某一个节气的主宰。土是相位缓冲、承载介质与转化层。

## 4. 节气交界

出生在节气交界前后，尤其是前后 24 至 72 小时内，常带有双节气结构。

例如：

- 谷雨末、立夏初：木火交界，积累转为显化；
- 大暑末、立秋初：爆火转初金，过热中出现终局判断；
- 霜降末、立冬初：冻金入水，裁断之后开始深藏；
- 大寒末、立春初：冰库孕木，长期压缩后开始破局。

## 5. 替换旧月支系统

```text
旧结构：月支 / 月度相位 = 年度主语
新结构：节气令 = 年度主语
```

旧的十二月支与十二个月度相位可以保留为历史索引，但不再作为现代命理学的主要解释层。

## 6. 判断公式

```text
人格季节底色 =
出生节气令
+ 节气交界状态
+ 出生地纬度与气候
+ 出生时地方太阳时
+ 出生月相
+ 孕期季节路径
+ 家庭与时代现实变量
```

其中，节气令是年度太阳相位的主轴。它描述人长期如何理解世界、整理经验、选择行动方向与处理阶段变化。
