# 144 月度相位类型（旧版历史入口）

本目录属于旧版结构：

```text
十二个月度季节相位 × 十二日内光热时段 = 144 种人格结构
```

该结构已经被新版节气光热体系替代。

新版主轴为：

```text
二十四节气令长期认知行为偏好 × 十二日内光热行动入口 = 288 种人格结构
```

## 旧版为何降级

旧版使用十二个月度相位，粒度较粗，并且依赖传统元素语言。

新版做了三处主要修正：

1. 年度端从十二个月度相位升级为二十四节气令。
2. 年度端解释对象从旧式心理欲望表述改为长期认知行为偏好。
3. 类型命名改为“节气 · 日内时段”，例如 `夏至·日出`。

## 保留原因

本目录仅作为历史版本入口，便于追踪体系演化。当前仓库首页、Pages 页面、理论文档与生成器均以 288 节气光热体系为准。
