# 144 五行性格相位类型

本目录使用新版命名体系：

```text
月度季节五行底色 × 日内光热五行底色 = 144 种人格结构
```

## 生成方式

完整 144 个类型文档由生成器生成：

```bash
python tools/generate_144_personality_docs.py
```

GitHub Pages 部署流程也会在上传 `docs/` 前自动运行该生成器。

## 理论主语

- 月度端：一年中的季节相位、温度趋势、光照趋势与五行性格。
- 日内端：地方太阳时、日出、太阳高度、温度滞后、日落、入夜、最低温与破晓。
- 土在月度系统中主要表示前后季节之间的连接、孕育、缓冲与向前转化。
- 土在日内系统中保留性格化含义，例如先知土木、残照土金、躁动土金。
