# 照庭·守界型

> 类型编码：午月 × 戌时  
> 底层关系：三合共振  
> 判读公式：月支 = 欲望模式底色；时支 = 思维模式偏好底色。

## 1. 月支欲望模式底色：午月｜照庭

中心显赫型欲望：欲望指向被看见、被承认、直接表达、价值照耀和舞台中心。它天然追求高光与清晰的正面存在。

在现代命理学里，月支不直接代表现实财富、职业或命运等级，而是代表出生季节相位压入人格后的“想要方式”。也就是说，它描述一个人如何生成欲望、如何感到世界值得追求、如何判断什么东西有吸引力。

## 2. 时支思维模式偏好底色：戌时｜守界

守界型思维：偏好风险隔离、保守校验、清理余烬、维护秩序边防。

时支不只是行动方式，也不是简单的外在表现。它更适合被定义为思维入口：人如何组织信息、如何判断可行性、如何启动推演、如何把欲望翻译成路径。

## 3. 两种底色的交互机制：三合共振

两种底色进入同一气脉，欲望与思维会向同一元素场汇流。优势是叙事统一、路径感强；风险是被同一套解释吸走，降低对其他变量的敏感度。

具体到本型，午月的欲望会不断提供“目标感”和“价值感”，而戌时的思维会决定这些目标如何被解释、拆分、审查和落地。两者不是静态相加，而是一个持续循环：欲望提出方向，思维选择路径；路径反过来修正欲望的尺度、温度与边界。

## 4. 最终人格表现

最终表现为：以照庭式欲望选择目标，以守界式思维处理路径。面对外界时，通常先由午月欲望决定“什么值得要”，再由戌时思维决定“怎样理解、拆解、推进或拒绝”。在高阶状态下，此型能把显赫、照耀、极化与收束、燥化、守门、清理组合成稳定的方法论；在低阶状态下，则会表现为三合共振带来的偏执、迟滞、过热、过冷或模块分裂。

在人际与事业场景中，此型通常不会单纯表现为某个地支的刻板形象，而会表现为“欲望底色 + 思维偏好”的复合结构：想要的东西带有午月气机，理解世界的方式带有戌时气机。

## 5. 高阶用法

- 先识别欲望来自哪里：它是想扩张、保全、显赫、裁断、回源，还是想建立容器。
- 再识别思维如何处理欲望：它会直接推进、延迟蓄养、技术化、标准化、博弈化，还是潜入整体感。
- 最后调整顺序：让欲望负责定方向，让思维负责建路径，避免思维过早否决欲望，也避免欲望压倒思维。

## 6. 核验

本文件只讨论世界观底色、欲望生成方式、思维偏好和二者互动机制；不把任何组合直接等同于财富、职业、婚恋、疾病、寿夭或固定命运。它保留传统地支的动态精神动力学精华，同时进入现代命理学“自然节律—精神结构—现实控制变量分离”的框架。
