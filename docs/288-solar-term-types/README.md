## 理论主语

- 年度端：二十四节气令，描述长期认知行为偏好。
- 日内端：地方太阳时、日出、正午、午后高热、日落、夜初、深夜、黎明前等光热时段。
- 旧版传统元素表述只作为历史解释层，不再作为当前体系主语。

## 秋分示例

```text
总结、分割、衡量、归类、设定边界、建立对称关系
```

所以 `秋分·日落前` 会形成强烈的分割—裁断同向结构：长期认知上倾向总结归类，行动入口上倾向边界判断与阶段结算。
