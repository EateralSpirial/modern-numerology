## 理论主语

- 年度端：二十四节气令，描述长期认知行为偏好。
- 日内端：地方太阳时、日出、太阳高度、温度滞后、日落、入夜、最低温与破晓。
- 土：作为节气交界、状态承载、相位缓冲和转化介质。
- 旧版“欲望生成方式”相关表述被替换为“长期认知行为偏好”。

## 秋分示例

秋分令的核心长期认知行为偏好是：

```text
总结、分割、衡量、归类、设定边界、建立对称关系
```

所以“秋分令 × 裁断金”会形成金气同频：长期认知上倾向总结归类，行动入口上倾向边界判断与阶段结算。
